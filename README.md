Personal Portfolio Website - Hasan raza

This is the complete source code for my personal portfolio website. It's a clean, single-page, fully responsive site built to showcase my skills, projects, and educational background as a tech enthusiast and developer.

The live version can be viewed here: kishlayprakhar.github.io/portfolio/ (Note: You may need to update this link)


Fully Responsive: Designed to look great on all devices, from mobile phones to desktops.

Single-Page Layout: Smooth scrolling navigation that directs users to different sections of the page.

Clean & Modern UI: A "Cute Dark Mode" theme with soft lavender and mint accents for a stylish, professional look.

Dynamic Header: A sticky navigation bar that stays at the top of the page for easy access.

Interactive Elements: Hover effects, gradient text, and glowing buttons to create an engaging user experience.

Project Structure

The website is divided into five main sections:

Home: A "hero" section with a profile picture, a greeting, and a call-to-action button.

About Me: A brief introduction, my passion for technology, and a professional image.

Education & Skills: A two-column layout displaying my educational qualifications and technical skills.

My Projects: A grid of project cards, each with an image, description, and links to the live demo and GitHub repository.

Get In Touch (Contact): A clear call-to-action with a "Say Hello" mail-to link and social media icons (GitHub, LinkedIn, Twitter).
